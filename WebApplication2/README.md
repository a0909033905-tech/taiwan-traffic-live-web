# 全台即時交通路況查詢系統

簡短說明：本系統使用政府開放資料「警廣即時路況」，資料來源為警察廣播電臺即時路況資料，內容包含事故、壅塞、施工、交通障礙等路況事件，並將資料整理為卡片式網頁，提供使用者查詢與篩選。

---

## 1. 專案介紹

專案名稱：全台即時交通路況查詢系統

這是一個以教學 / 作業為目的所開發的 ASP.NET Core MVC 網站，展示如何以 Controller 與 Razor View 顯示路況事件、做出篩選介面與卡片式的顯示樣式。本系統使用政府開放資料「警廣即時路況」作為資料來源。

## 2. 專案目的

- 練習 ASP.NET Core MVC（Controller、View、Model）流程
- 示範如何在前端提供地區與事件類型篩選功能
- 已串接政府開放資料「警廣即時路況」，並將資料轉成系統可顯示的事件模型。

## 3. 主要功能

- 顯示即時交通路況事件清單
- 依地區篩選：全部、北部、中部、南部、東部
- 依事件類型篩選：全部、事故、壅塞、施工、交通障礙
- 每筆事件顯示：事件類型、地區、道路名稱、方向、地點、發生時間、說明、生活建議
- 卡片式 UI，使用 Bootstrap 做出響應式版面

## 4. 使用資料來源

本系統使用政府開放資料「警廣即時路況」（警察廣播電臺即時路況資料）作為資料來源。資料包含事故、阻塞、道路施工、交通障礙、交通管制等即時路況事件。系統會將即時路況整理成卡片式網頁並提供地區與事件類型篩選，方便使用者查詢與快速掌握路況。若 API 暫時無法連線，系統才會使用備援資料以避免網頁無法顯示。

## 5. 使用技術

- .NET 10
- ASP.NET Core MVC（Controllers + Razor Views）
- Bootstrap（前端樣式）
- C#（Models / Controllers）

## 6. 專案架構（重點檔案）

- Controllers/TrafficController.cs — 提供 Index action 並處理篩選（從警廣即時路況取得資料並轉換）
- Models/TrafficEvent.cs — 事件模型（包含從警廣資料轉換的欄位）
- Views/Traffic/Index.cshtml — 路況列表與篩選 UI
- Views/Home/Index.cshtml — 自訂首頁（導向 /Traffic）
- Views/Shared/_Layout.cshtml — 全域 Layout（已修改網站標題與導覽連結）

（專案中還包含其他預設檔案，如 wwwroot、appsettings 等）

## 7. 使用方式

本專案可在 Visual Studio 或使用 dotnet CLI 執行：

1. 使用 Visual Studio 開啟解決方案並執行（F5）。
2. 或在專案根目錄（有 .csproj 的資料夾）執行：

   ```powershell
   dotnet restore
   dotnet run
   ```

3. 開啟瀏覽器，前往 Home 頁面（根路徑）或直接開啟:

   - /Traffic 或 /Traffic/Index — 查看即時交通路況（目前為假資料）

## 8. 生活應用情境

- 上班族查看通勤路段是否有事故或壅塞，以決定是否改道或改搭大眾運輸
- 物流司機或計程車司機規劃路線時，快速掌握沿途可能的施工或交通障礙
- 一般民眾在旅遊或假日出遊前，先行查詢主要道路的路況資訊
